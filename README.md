# Tea Shop Management Console App ☕

> A simple Java program to manage and display tea shop details using ArrayList, Iterator, and loops.

## 📦 Package: `javalearning`

This is a Java-based menu-driven console application that lets users:

- Insert multiple tea shop records
- Display the list using both `Iterator` and index-based loop
- Learn object-oriented concepts and Java collections through a real-world example

---

## 🚀 Features

- 📋 Add details of multiple tea shops dynamically
- 🔁 View tea shops using:
  - `Iterator`
  - `for` loop (index-based)
- 🧠 Practice with Java collections, object-oriented programming, and console I/O
